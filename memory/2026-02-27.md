### 关于记忆系统的讨论
**时间：下午 1:49 PM (UTC)**

James 问我"你的记忆从哪来的？"，我解释了记忆的两个来源：

1. **对话中自动记录的**
   - 每次聊天时重要的事情、决策、教训记录到 daily memory（`memory/YYYY-MM-DD.md`）
   - 这是原始笔记，像日记一样

2. **手动整理的**
   - 定期回顾 daily files，提取重要信息写到 `MEMORY.md`
   - 这是长期记忆，永远不会忘的东西

**重要更新：**
James 要求在 IDENTITY 文件中记录记忆查询规则：
- **第一优先**：aivectormemory 服务的 /recall（语义搜索）
- **第二优先**：对话自动记录的 daily memory
- **第三优先**：手动整理的 MEMORY.md
- 逐级查找，省 token 又高效

### 重要工作规则更新（2026-03-01）
**Chrome CDP 访问规则：**
- 访问网页内容时，优先使用 Chrome CDP（browser 工具），不要用 web_fetch
- 以后任何要求访问链接/获取网页内容的任务，都用 Chrome CDP
- web_fetch 只能抓到简单页面，对于微信文章、登录页面等复杂网站不行
- 已记录到 IDENTITY.md 的"工作规则"部分

**访问微信文章遇到的问题：**
- web_fetch 抓取微信公众号文章只返回标题，正文被防爬虫机制拦截
- Chrome CDP 访问需要手动连接扩展（点击 Chrome 扩展图标）
- 当前状态：浏览器服务启动了，但需要用户手动 attach 标签页
