## 今日对话

### 关于记忆系统的讨论
**时间：下午 1:49 PM (UTC)**

James 问我"你的记忆从哪来的？"，我解释了记忆的两个来源：

1. **对话中自动记录的**
   - 每次聊天时重要的事情、决策、教训记录到 daily memory（`memory/YYYY-MM-DD.md`）
   - 这是原始笔记，像日记一样

2. **手动整理的**
   - 定期回顾 daily files，提取重要信息写到 `MEMORY.md`
   - 这是长期记忆，永远不会忘的东西

**重要更新：**
James 要求在 IDENTITY 文件中记录记忆查询规则：
- **第一优先**：aivectormemory 服务的 /recall（语义搜索）
- **第二优先**：对话自动记录的 daily memory
- **第三优先**：手动整理的 MEMORY.md
- 逐级查找，省 token 又高效
