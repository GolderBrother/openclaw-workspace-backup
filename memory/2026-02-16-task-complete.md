## 孟健写作风格核心原则（10篇综合）

### 标题公式（10种）
1. `[情绪化开头] + [数字对比] + [行动表达]`
2. `[工具/方法] + [场景] + [数据对比] + [成本/效率优势]`
3. `[主题关键词] + [现象描述（权威+反常识+引用）]`
4. `[动作+结果+情绪] + [反问句（疑问+反常识+自豪）]`
5. `[工具/方法] + [数据对比] + [场景] + [承诺交付]`
6. `[好消息/趋势] + [核心信息] + [（数据验证）]`
7. `[产品/服务发布] + [时间承诺（从0到1）] + [资源数量]`
8. `[热点项目+数字冲击] + [个人投入+结果交付]`
9. `[个人行动] + [产品特点] + （[内容承诺]）`
10. `[个人投入+数量] + [产品+对比优势]`

### 开头模式（7种）
1. **结果前置+数字冲击+个人立场**
2. **钩子句+结论前置+数据轰炸+趋势描述**
3. **品牌锚点+震撼性现象描述**
4. **场景化观察+痛点列表+解决方案宣布**
5. **热点引入+数据轰炸+权威背书**
6. **个人体验+产品介绍+价值评价**
7. **个人体验+数量冲击+结果+承诺**

### 结构特征（10种）
- 编号从00开始
- 3条命令/3步法
- Day-by-Day时间线（具体日期）
- 场景化叙事（时间戳、手机操作）
- 流程图视觉化（箭头+缩进）
- 代码示例
- 成本表格对比
- 对比总结（编号列表）
- 本质区别（升华）
- 单一功能模块 / 组合功能模块

### 语言特点（8种）
- 数据轰炸（热度+投入+结果+成本+效率）
- 权威背书（媒体+KOL）
- 对比结构（不是...而是...）
- 真实感受（"说实话""这玩意儿太香了"）
- 推荐强调（"强烈推荐"）
- 个人喜好（"我觉得"）
- 强调关键词（真正/完整/随时/最强大）
- 数字量化（时间/数量/成本/效率）

### 稳定特征（10篇都有）
✅ 数字量化（标题和内容）
✅ 模块化结构（编号分段）
✅ 口语化表达（但不随意）
✅ 数据说话（具体且有冲击力）
✅ 对比手法（建立认知冲突）

### 孟健写作风格核心原则
1. **开场必须炸**：结果前置+数字冲击+个人立场
2. **对比必须狠**：过去vs现在、成本对比、工具vs团队
3. **数字必须具体**：不是"很多"，是"565个"、"6个"
4. **场景必须真实**：时间戳、手机操作、第三方身份
5. **门槛必须低**："一天搞定"、"3条命令"
6. **人群必须明确**：适合+不适合（反向筛选）
7. **理念必须升华**：不是替代，是放大
8. **收尾必须留白**：不强求，用不用是你的事
