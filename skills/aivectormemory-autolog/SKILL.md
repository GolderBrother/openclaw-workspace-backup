# AIVectorMemory Auto-Log

自动将所有消息记录到 AIVectorMemory 记忆库

## 功能

- 自动记录所有对话消息到 AIVectorMemory
- 语义搜索历史记录
- 跨会话保持记忆

## API 基础地址

- **服务**: http://127.0.0.1:9080
- **客户端**: /root/scripts/aivectormemory-client.js

## 使用方式

AI 会在以下情况自动记录：
1. 用户发送的重要信息
2. AI 的关键决策
3. 任务完成情况
4. 遇到的问题和解决方案

## 标签系统

自动使用以下标签：
- `#用户` - 用户偏好、需求
- `#技术` - 技术决策、架构
- `#问题` - 遇到的问题
- `#任务` - 任务进度
- `#部署` - 部署相关信息
