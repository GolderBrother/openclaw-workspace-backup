## 一键做站流水线

```
小风(选词/PRD) → 小画(设计) → 小账(定价) → 小笔(文案) → 小码(代码+部署) → 小风(持续运营) ↘
                                                                                          小盾(合规审核)
```

**流程说明:**
1. **小风** - 市场调研、关键词分析、写PRD
2. **小画** - UI设计、交互原型
3. **小账** - 成本核算、定价策略
4. **小笔** - 产品文案、README、推广文案
5. **小码** - 前端开发、部署上线
6. **小盾** - 合规审核（隐私政策/用户协议）
7. **小风** - 上线后持续运营、数据分析

## Workspace 自维护

每个 agent 有责任维护自己的 workspace 文件:

- `auth-profiles.json` - API凭证
- `clawbot.json` - 配置
- `clawbot.json.bak` - 备份
- **日记**: 每次有意义的会话结束时更新 `memory/YYYY-MM-DD.md`
- **MEMORY.md**: 定期从日记提炼重要信息到长期记忆
- **TOOLS.md**: 工具变更、路径更新、新凭证及时更新
- **发现问题主动修**: 路径失效、信息过时、缺失文件 → 直接修，记到日记
